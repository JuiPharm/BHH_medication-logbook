# 🏥 สมุดยาฝาก (Medication Logbook)

ระบบบันทึกยาฝากพร้อม OCR อ่านใบสั่งยา ใช้งานผ่าน GitHub Pages + Google Sheets

## 🔗 Links
- **Live App**: https://YOUR_USERNAME.github.io/medication-logbook
- **Database**: Google Sheets

## ⚙️ Setup
1. สร้าง Google Sheets ตามโครงสร้างในเอกสาร
2. Deploy Google Apps Script แล้วแก้ `js/config.js`
3. เปิดใช้ GitHub Pages

## 🛠️ Tech Stack
- Frontend: HTML5 + Bootstrap 5 + Vanilla JS
- OCR: Tesseract.js
- Backend: Google Apps Script
- Database: Google Sheets
- Hosting: GitHub Pages
